Colours

Corporate consistency is good for branding, e.g. the standard blues, and they are used as base colours for fonts and are useful as backgrounds:

.*-iansblue-dark

.*-iansblue-light

(* represents the multiple prefixes tailwind adds custom colours to, e.g. "text" or "bg").

However, we are producing educational content and that requires a wider range of colours to explain concepts and make it interesting.

Consistency of colour use is still essential within courses. Using the same colours across courses constrains the designer too much and starts to look boring.

Course specific style guides

The solution is to have standardised base colours for each course. This framework is built to enable that. Here are some examples:

Breadcrumb

Use .ians-breadcrumb to style the breadcrumb. The breadcrumb aims to help learners quickly place the current page within the course. It only needs to be used in longer courses or courses with nested levels.
