I was nesting lists wrong!

Old way

<ul>
    <li>Bad</li>
    <ul>
        <li>Notice this list is on its own line</li>
        <li>Bad</li>
    </ul>
    <li>Bad</li>
    <li>Bad</li>
</ul>

Correct way

<ul>
    <li>Good</li>
    <li>Gooderrer:
        <ul>
            <li>The nested list should be inside a parent list item with some text to introduce the nested list.</li>
            <li>Good</li>
        </ul>
    </li>
    <li>Good</li>
    <li>Good</li>
</ul>
